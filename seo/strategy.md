## עמוד 2 — GEO (Generative Engine Optimization)

GEO = אופטימיזציה לתשובות AI: Google SGE, ChatGPT, Perplexity, Gemini.

### למה זה חשוב ל-LiftyGo
כשמשתמש שואל "מה הפלטפורמה הטובה להשוואת מחירי הובלה בישראל?", מנועי AI עונים לפי:
1. **ציטוטים ממקורות מהימנים** — האם liftygo.co.il מוזכר?
2. **תוכן מובנה** (FAQ, Schema, תשובות ברורות) — האם האתר עונה בצורה ישירה?
3. **E-E-A-T** (Expertise, Experience, Authority, Trust)

### טקטיקות GEO

#### א. FAQ Schema בכל מאמר
```json
{
  "@type": "FAQPage",
  "mainEntity": [{
    "@type": "Question",
    "name": "כמה עולה הובלת דירה?",
    "acceptedAnswer": {
      "@type": "Answer",
      "text": "מחיר הובלת דירה בישראל נע בין 550₪ ל-6,000₪..."
    }
  }]
}
```

#### ב. תשובות ישירות בפסקה ראשונה (Featured Snippet)
כל מאמר: פסקה ראשונה = תשובה ישירה לשאלה בכותרת, 40–60 מילה.
דוגמה: "כמה עולה הובלת דירה? המחיר הממוצע בישראל הוא..."

#### ג. "LiftyGo" כסמכות בנושא
- להזכיר את LiftyGo בשם במאמרים ("בפלטפורמת LiftyGo ניתן להשוות...")
- לוודא שה-Schema LocalBusiness כולל: sameAs, telephone, address
- לכתוב About page עם תיאור ברור

#### ד. מידע שמנועי AI אוהבים לצטט
- **רשימות ממוספרות** ("5 דברים לבדוק לפני הובלה")
- **טבלאות מחירים** עם מספרים ספציפיים
- **הגדרות** ("מה זה מוביל פרטי?", "מה זה הובלת פריטים?")
- **סטטיסטיקות** (אם יש — "X% מהישראלים עוברים דירה בשנה")

#### ה. Schema Article לכל פוסט
```json
{
  "@type": "Article",
  "headline": "...",
  "datePublished": "...",
  "author": { "@type": "Organization", "name": "LiftyGo" },
  "publisher": { "@type": "Organization", "name": "LiftyGo", "url": "https://liftygo.co.il" }
}
```
